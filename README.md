# codexrepository

一个使用 **HTML + CSS + JavaScript** 实现的简易贪吃蛇小游戏。

## 文件说明

- `index.html`：页面结构与游戏入口。
- `style.css`：页面与游戏画布样式。
- `script.js`：游戏核心逻辑（移动、碰撞检测、计分、最高分存储）。

## 运行方式

### 方式一：直接打开
1. 在文件管理器中进入仓库目录。
2. 双击 `index.html`，使用浏览器打开即可开始游戏。

### 方式二：本地静态服务器（推荐）
如果你安装了 Python 3，可在仓库根目录运行：

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000/index.html
```

## 游戏操作

- 点击“开始游戏”开始。
- 使用键盘方向键 `↑ ↓ ← →` 控制蛇移动。
- 吃到食物加分，撞墙或撞到自己则游戏结束。
- 点击“重新开始”可立即开新局。
